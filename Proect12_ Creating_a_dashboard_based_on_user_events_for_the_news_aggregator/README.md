## Создание дашборда по пользовательским событиям для агрегатора новостей

## Задачи проекта
Анализ пользовательского взаимодействия с карточками статей. Создание дашборта по техническому заданию и презентации.

## Описание проекта
Подготовлен интерактивный дашборд на основе данных о взаимодействии пользователей с карточками статей Дзена. Для создания дашбордов использован BI-инструмент Tableau.

## Навыки и инструменты
Tableau, построение дашбордов, продуктовые метрики, создание презентации Power Point.
